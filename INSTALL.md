# üì¶ Quick Installation Guide - EcoCheck

## Step 1: Get Gemini API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the generated API key (you'll need it in step 3)

## Step 2: Load the Extension in Chrome

1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Enable **"Developer mode"** (toggle in the top right corner)
4. Click **"Load unpacked"**
5. Select the complete `EcoCheck` folder
6. The extension should appear in your extensions list

## Step 3: Configure the API Key

1. Click the **EcoCheck** icon in Chrome's toolbar
2. If you don't see the icon, click the extensions icon (puzzle) and pin EcoCheck
3. In the popup that appears, enter your Gemini API key
4. Click **"Save"**

## Step 4: Use EcoCheck!

1. Navigate to any product page (Amazon, online stores, etc.)
2. Click the EcoCheck icon
3. Click **"Analyze Product"**
4. Wait a few seconds
5. View the environmental analysis results!

## üé® Add Icons (Optional)

If you want custom icons:

1. Create three images:
   - `icon16.png` (16x16 pixels)
   - `icon48.png` (48x48 pixels)
   - `icon128.png` (128x128 pixels)

2. Place them in the `assets/` folder

3. Reload the extension in `chrome://extensions/`

### "Cannot load extension"
- Make sure to select the complete `EcoCheck` folder, not individual files
- Verify that all files are present

### "Error analyzing product"
- Verify that your API key is correct
- Make sure you're on a product page
- Check the browser console (F12) for more details

### "No data extracted"
- Some pages may have unique HTML structures
- Try on different product websites
